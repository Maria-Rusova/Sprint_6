# Sprint_6
### UI‑тестирование сайта «Яндекс Самокат»
    Автоматизированные UI‑тесты функциональности веб‑сайта сервиса «Яндекс Самокат», выполняемые в браузере Mozilla Firefox.

Стек технологий
Язык программирования: Python 3.14;
Фреймворк для тестирования: PyTest;
Инструмент автоматизации: Selenium;
Архитектура тестов: Page Object Model (POM);
Отчётность: Allure;
Браузер и драйвер: Mozilla Firefox + geckodriver.

    Реализованные сценарии тестирования
1. Тестирование домашней страницы:
- проверка работы выпадающего списка в разделе «Вопросы о важном» (клик на каждый вопрос);
- проверка перехода на страницу заказа через кнопки в шапке и основной секции главной страницы;
- проверка перехода на страницу Дзен при клике на логотип «Яндекс» (в шапке и основной секции);
- - проверка возврата на главную страницу при клике на логотип «Самокат» (в шапке и основной секции).

2. Тестирование страницы заказа самоката:
- полный цикл оформления заказа с двумя наборами тестовых данных (переход с главной страницы по кнопке «Заказать»), включая проверку сообщения «Заказ оформлен»;
- оформление заказа до появления сообщения «Заказ оформлен» (переход с главной страницы по кнопке «Заказать»).

    Команды для запуска

Запуск тестов в один поток (подробный вывод): `pytest -v`

Запуск тестов с сохранением результатов для Allure: `pytest tests/ --alluredir=allure_results`

Формирование отчёта Allure: `allure serve allure_results`
